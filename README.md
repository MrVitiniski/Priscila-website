# 💄 Priscila Silva - Harmonização Facial

**Um website focado em conversão e elegância, criado para apresentar serviços de estética facial e transformar visitantes em agendamentos clínicos.**

Este projeto foi desenvolvido com uma estética minimalista e luxuosa, utilizando tons pastéis e efeitos de transparência para transmitir a sofisticação e o cuidado que o público de harmonização facial busca.

## ✨ Destaques do Projeto

- **Foco em Conversão (CRO):** Estrutura de "Linktree" aprimorada com seções de serviços integradas, facilitando o contato direto via WhatsApp.
- **Design Minimalista:** Uso de *Glassmorphism* e paleta de cores "nude" para um visual limpo, moderno e profissional.
- **Menu Mobile-First:** Navegação otimizada para smartphones, com ícones intuitivos e acesso rápido às redes sociais.
- **Catálogo de Procedimentos:** Seção detalhada de serviços com cards elegantes e descrições claras para o paciente.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica focada em SEO.
- **CSS3**: Estilização personalizada, degradês suaves e efeitos de desfoque (*backdrop-filter*).
- **JavaScript**: Lógica de interação para os menus e transições.
- **Font Awesome**: Ícones minimalistas para identificação de serviços.
- **Google Fonts**: Tipografia refinada (Poppins & Playfair Display).
